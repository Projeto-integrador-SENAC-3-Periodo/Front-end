# Sistema de gerenciamento de atividades complementares - SENAC

## Sobre o projeto  
Este projeto é uma aplicação front-end desenvolvida com React + TypeScript, com o objetivo de gerenciar atividades complementares de alunos, permitindo o controle de horas e acompanhamento de solicitações de forma clara e intuitiva.

O sistema foi desenvolvido como parte do Projeto Integrador da Faculdade SENAC, com foco em resolver uma demanda real do ambiente acadêmico.

A aplicação foi construída com foco em organização, reutilização de componentes e uma interface moderna e responsiva.

---

## Objetivo  
Facilitar o acompanhamento das atividades complementares para alunos, coordenadores e administradores da Faculdade SENAC, proporcionando maior controle, organização e transparência no registro e validação das horas acadêmicas.

---

## Dependências essenciais  

Para rodar o projeto corretamente, basta executar:

npm install  

Esse comando instala automaticamente todas as dependências necessárias.

Caso seja necessário instalar manualmente:

Dependências principais:  
npm install react react-dom axios @tanstack/react-query  

Ambiente de desenvolvimento:  
npm install -D vite typescript @types/react @types/react-dom  

Estilização (necessária para o layout):  
npm install tailwindcss postcss autoprefixer tailwindcss-animate

## Como rodar o projeto  

1. Clone o repositório  
git clone <url-do-repositorio>  

2. Acesse a pasta  
cd Front-end-main  

3. Instale as dependências  
npm install  

4. Execute o projeto  
npm run dev  

Acesse no navegador:  
http://localhost:5173
