# 🎓 Sistema de Gerenciamento de Atividades Complementares (Front-end) - SENAC

---

## 📌 Sobre o projeto  
Este projeto é uma aplicação front-end desenvolvida com **React + TypeScript**, com o objetivo de gerenciar atividades complementares de alunos, permitindo o controle de horas e o acompanhamento de solicitações de forma clara e intuitiva.

O sistema foi desenvolvido como parte do **Projeto Integrador da Faculdade SENAC**, com foco em atender uma demanda real do ambiente acadêmico.

A aplicação foi construída priorizando organização, reutilização de componentes e uma interface moderna e responsiva.

---

## 🎯 Objetivo  
Facilitar o acompanhamento das atividades complementares para **alunos, coordenadores e administradores** da Faculdade SENAC, proporcionando maior controle, organização e transparência no registro e validação das horas acadêmicas.

---

## 🚀 Tecnologias utilizadas  

- ⚛️ React  
- 🟦 TypeScript  
- ⚡ Vite  
- 🎨 Tailwind CSS

- 🌐 Axios


3. Instale as dependências

4. Execute o projeto
npm run dev
